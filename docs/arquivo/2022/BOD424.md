## Transcrição

Quadrinho com uma pessoa numa canoa usando uma garrafa grande para remar.

Sub-legenda:
> REMOAGEM

## Explicação

O *remuage* é uma técnica difundida em Champagne para outras regiões vinícolas que consiste em girar a garrafa sistematicamente para que as leveduras se direcionem do fundo para o gargalo da garrafa. Não há tradução definitiva para português, com alguns autores usando *remuagem* ou *remoagem*. O movimento é semelhante àquele da prática de remo.
