## Transcrição

Fluxograma: **O VINHO É DO JURA?**

*   **Não** -> **CAI FORA!**
*   **Sim!** -> **QUE COR?**
    *   **Branco** -> **Lembra o caju amigo do saudoso Pandoro?**
        *   **Não** (Tem cheiro de armário de avó!) -> **SAVAGNIN TRADITION OU VIN JAUNE**
        *   **Sim** -> **CHARDONNAY E/OU SAVAGNIN TRADITION**
    *   **Branco** -> **Tem cheiro de cocô** -> **CHARDONNAY OUILLÉ** (ou talvez defeito?)
    *   **Tinto** -> **Tá imundo mas tá legal** -> **PINOT NOIR** (ou Poulsard/Trousseau?)
        *   **Tá Delícia** -> **TROUSSEAU**
    *   **Rosé** -> **Tem Certeza?** -> **Não** -> **Tinto** (Volta para Tinto - piada com a cor clara dos tintos de lá)
        *   **Sim** -> **CORAIL** ou **POULSARD**
    *   **Não sei** -> **TÁ ESCRITO OVERNOY NO RÓTULO?**
        *   **Não!** -> **Tem Gnomo?** -> **Sim** -> **DOM. DE L'OCTAVIN! PRBNS!**
        *   **Não!** -> **É Doce?** -> **Sim** -> **MACVIN**
            *   **Não** -> **É UM GANEVAT BAGUNCINHA**
        *   **Sim!** -> **OVERNOY HOUILLON** -> **POSTA NO INSTA, SUA DIVA INCONTROLÁVEL!**
        *   **Sim!** -> **OVERNOY-CRINQUAND** -> **SEU BURRO, VOCÊ CONFUNDIU O OVERNOY!**

## Explicação

Um guia para os vinhos "difíceis" e *hypados* da região do **Jura** (França).
*   **Oxidação:** O "cheiro de armário de avó" (naftalina/velho) e "caju" (oxidação controlada, estilo *Sous Voile*) são típicos dos vinhos locais (**Vin Jaune**, **Savagnin**). "Pandoro" era uma famosa doceria/bar de SP que servia o "Caju Amigo".
*   **Redução:** "Cheiro de cocô" refere-se à redução extrema, comum em vinhos naturais (como alguns Chardonnays de lá).
*   **Tintos Claros:** A piada "Rosé -> Tem certeza? -> Não -> Tinto" brinca com o fato de que os tintos de **Poulsard** são tão claros que parecem rosés.
*   **Produtores Cult:**
    *   **Pierre Overnoy (agora Houillon):** O "Deus" do vinho natural. Quem bebe "posta no Insta" para ostentar.
    *   **Overnoy-Crinquand:** Outro produtor com mesmo sobrenome, mas muito menos famoso. Quem compra achando que é o Pierre Overnoy é chamado de "burro".
    *   **Domaine de l'Octavin:** Famoso pelos rótulos com gnomos.
    *   **Ganevat:** Outro gigante, mas com tantos rótulos diferentes que vira uma "baguncinha".
