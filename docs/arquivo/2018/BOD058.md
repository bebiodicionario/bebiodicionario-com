## Explicação

Um mapa tipográfico de Portugal onde os nomes das principais regiões vinícolas (Dão, Douro, Alentejo, etc.) são arranjados visualmente para representar suas localizações geográficas aproximadas, formando o contorno do país. O termo "Cartotipografia" é um neologismo que une cartografia e tipografia.
