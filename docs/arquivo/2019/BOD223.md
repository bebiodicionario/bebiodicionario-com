## Transcrição

A imagem é um cartum dividido em duas partes, comparando tipos de podas de videira com passageiros de metrô.

*   **Parte Superior:** Desenhos técnicos de três sistemas de condução de videiras:
    1.  **ROYAT SIMPLES:** Um tronco com um braço horizontal para a esquerda.
    2.  **GUYOT DUPLO:** Um tronco com dois braços horizontais (um para cada lado).
    3.  **GOBELET:** Um tronco curto com galhos saindo para cima em forma de taça/arbusto.

*   **Parte Inferior:** Uma cena de vagão de metrô lotado, intitulada "**METRÔ ÀS 18:10 DE QUARTA-FEIRA**". Os passageiros seguram nos apoios superiores imitando os formatos das videiras acima:
    *   Uma pessoa segura com um braço só (Royat).
    *   Outra segura com os dois braços abertos (Guyot).
    *   Outra está com os braços para cima em confusão (Gobelet).

Rodapé: "**SISTEMAS DE CONDUÇÃO DE VIDEIRAS**".

## Explicação

O post tem objetivo **didático** através do humor, usando uma analogia visual do cotidiano urbano para explicar conceitos de viticultura.

1.  **Sistemas de Condução:** Explica visualmente como a videira é podada e conduzida nos arames.
    *   *Royat (Cordon):* Permanente, braços fixos.
    *   *Guyot:* Renovável, varas novas a cada ano.
    *   *Gobelet:* Em forma de arbusto, sem arames (comum em vinhas velhas ou regiões quentes).
2.  **A Analogia:** Compara a forma física da planta com a posição desconfortável dos passageiros se segurando no metrô em horário de pico, facilitando a memorização para o público "urbanóide" (da cidade grande) que talvez nunca tenha pisado em um vinhedo.
