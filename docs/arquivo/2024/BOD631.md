## Legenda

Estou no Tinder só para espalhar a palavra do Skin Contact.


## Transcrição

Ilustração de uma mulher com expressão meio desesperada/triste, diante de uma garrafa de vinho laranja.
Texto: **"SEJA VINHO LARANJA TENHA CONTA-TINHOS"**.

## Explicação

Trocadilho triplo:
1.  **Vinho Laranja** (Orange Wine).
2.  O termo técnico para a produção de vinho laranja é **"Skin Contact"** (contato com as cascas).
3.  **"Contatinhos"** (gíria para parceiros casuais/dates).
A piada sugere "ter contato com a casca" (skin contact) como metáfora para ter "contatinhos" (contato físico/relacionamento).
